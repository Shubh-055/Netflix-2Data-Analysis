# 🎬 Netflix Data Analysis Project

## 📌 Project Overview

This project analyzes Netflix data using Excel, Python, and Power BI to uncover insights about content trends, genres, and distribution.

---

## 🛠 Tools Used

* Excel (Data Cleaning & Basic Analysis)
* Python (Pandas for data processing)
* Power BI (Dashboard & Visualization)

---

## 📊 Key Insights

* Movies dominate the platform compared to TV Shows
* Drama and International genres are most popular
* Significant growth in content after 2015
* United States produces the highest number of titles
